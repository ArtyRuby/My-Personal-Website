## 🖼️ 如何添加项目配图

### 方法 1：使用 Finder（推荐，最简单）

1. **打开 Finder**，按 `Cmd + Shift + G`
2. **输入路径**：`/Users/mi/cursor/resume/assets/images/projects`
3. **拖拽图片**：将你的项目图片拖拽到这个文件夹
4. **重命名文件**：确保文件名与下方列表一致

**需要的图片文件名：**
- `meland.jpg` - Meland 品牌联名项目
- `pla-trial.jpg` - PLA Trial 新品
- `spring-combo.jpg` - 新春耗材 Combo
- `jisulife.jpg` - Jisulife 品牌运营
- `xiaomi-cloud.jpg` - 小米云服务
- `marketing-case.jpg` - 纤启新浪潮策划案

**图片规格建议：**
- 尺寸：至少 800x600px
- 格式：JPG、PNG、WebP 均可
- 比例：16:9 或 4:3
- 大小：每张不超过 2MB

### 方法 2：使用终端命令

```bash
# 进入项目图片文件夹
cd /Users/mi/cursor/resume/assets/images/projects

# 列出当前文件
ls -la

# 复制图片到这里（替换为你的图片路径）
cp ~/Desktop/你的图片.jpg ./meland.jpg
```

## 🎨 图片如果没有怎么办？

**不用担心！** HTML 已经设置了智能回退机制：

- 如果图片文件不存在，会自动显示 emoji 占位符（🏢、🎨、🎁 等）
- 如果头像不存在，会自动显示紫色字母 "R" 的 SVG 占位符
- 网站依然可以正常运行，只是视觉效果会稍微逊色

## 🔧 常见问题

### Q1: 图片显示不出来？

**检查清单：**
1. 文件名是否完全一致（包括大小写）？
2. 文件扩展名是否正确（.jpg 不是 .jpeg）？
3. 文件是否在正确的文件夹中？

**快速检查命令：**
```bash
# 查看项目图片文件夹内容
ls -la /Users/mi/cursor/resume/assets/images/projects/

# 查看头像是否存在
ls -la /Users/mi/cursor/resume/assets/images/avatar.jpg

# 查看简历是否存在
ls -la /Users/mi/cursor/resume/assets/pdf/resume.pdf
```

### Q2: 简历下载按钮点击没反应？

1. 确认 `resume.pdf` 文件存在
2. 尝试右键点击按钮 → "链接另存为"
3. 检查浏览器是否阻止了下载

### Q3: 想用 PNG 格式的图片可以吗？

**可以！** 只需修改 HTML 中的文件扩展名：

例如，如果你的 Meland 项目图是 PNG 格式：
```html
<!-- 原来 -->
<img src="assets/images/projects/meland.jpg" ...>

<!-- 改为 -->
<img src="assets/images/projects/meland.png" ...>
```

### Q4: 如何更换 LinkedIn 链接？

在 `index.html` 文件中搜索 `https://linkedin.com`，替换为你的真实 LinkedIn 链接。

## 📤 如何部署到线上

### 方法 1：Vercel（推荐）

1. 访问 [vercel.com](https://vercel.com)
2. 注册账号并连接 GitHub
3. 将项目上传到 GitHub 仓库
4. 在 Vercel 导入项目
5. 自动部署，获得免费域名

### 方法 2：Netlify

1. 访问 [netlify.com](https://netlify.com)
2. 注册账号
3. 拖拽整个 `resume` 文件夹到部署区域
4. 自动部署，获得免费域名

### 方法 3：GitHub Pages

1. 创建 GitHub 仓库（命名为 `username.github.io`）
2. 上传所有文件到仓库
3. 在仓库 Settings → Pages 中启用
4. 访问 `https://username.github.io`

## 💡 优化建议

### 图片优化
```bash
# 使用 ImageMagick 批量压缩图片（需要先安装）
brew install imagemagick

# 压缩图片到 80% 质量
magick mogrify -quality 80 -resize 1200x /Users/mi/cursor/resume/assets/images/projects/*.jpg
```

### 转换为 WebP 格式（更小体积）
```bash
# 安装 cwebp
brew install webp

# 转换图片
cwebp -q 80 meland.jpg -o meland.webp
```
