## 🚨 常见问题

### Q: 图片不显示怎么办？

**检查清单：**
1. ✅ 文件名是否完全一致（包括大小写）？
2. ✅ 文件扩展名是否正确（.jpg 不是 .JPG）？
3. ✅ 文件是否在正确的文件夹中？

**快速修复：**
```bash
# 查看文件名（注意大小写）
ls -la /Users/mi/cursor/resume/assets/images/projects/

# 批量重命名为小写（如果需要）
cd /Users/mi/cursor/resume/assets/images/projects/
for f in *; do mv "$f" "$(echo $f | tr '[:upper:]' '[:lower:]')"; done
```

### Q: 我的图片是 PNG 格式，可以用吗？

**可以！** 有两种方法：

**方法1：转换为 JPG（推荐）**
```bash
# 使用 macOS 自带的 sips 工具
sips -s format jpeg 你的图片.png --out meland.jpg
```

**方法2：修改 HTML 代码**

将 HTML 中对应的 `.jpg` 改为 `.png`

### Q: 如何优化图片大小？

```bash
# 使用 sips 压缩图片
sips -Z 1200 --setProperty formatOptions 80 /Users/mi/cursor/resume/assets/images/projects/*.jpg

# 或使用 ImageMagick（需先安装：brew install imagemagick）
magick mogrify -resize 1200x -quality 80 /Users/mi/cursor/resume/assets/images/projects/*.jpg
```

## 🎨 图片规格建议

### 头像（avatar.jpg）
- **尺寸**：500x500px ~ 1000x1000px
- **格式**：JPG、PNG
- **大小**：< 1MB
- **内容**：清晰的正面照，专业形象

### 项目配图（6张）
- **尺寸**：800x600px ~ 1920x1080px
- **比例**：16:9 或 4:3
- **格式**：JPG、PNG、WebP
- **大小**：每张 < 2MB
- **内容**：项目相关的高质量图片

### 简历 PDF
- **格式**：PDF
- **大小**：< 5MB
- **页数**：1-2 页为佳
